####################################################################################
#INSTRUCTIONS
####################################################################################

After downloading the project,

  * open your preferred terminal & 'cd' into the folder containing the project
  * use the command 'node server.js' to run the server
  * wait for the terminal to print 'Server is running on port 3000'
  * using your preferred web browser, type 'localhost:3000' into the address bar
  * after pressing enter, the web application will be displayed and ready to test
